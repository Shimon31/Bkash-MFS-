# 💸 MFS bKash Clone App (Kotlin + Firebase)

A Mobile Financial Service (MFS) Android application built using **Kotlin** and **Firebase**.  
This app simulates a simplified version of services like bKash, supporting **User, Agent, and Admin** roles with secure money transfer functionality.

## 🚀 Features

### 👤 User Panel
- Send money to other users
- Cash out via agents
- View transaction history
- Firebase Authentication login

### 🧑‍💼 Agent Panel
- Receive cash-in from admin
- Cash-out requests from users
- Manage wallet balance
- Transaction tracking

### 🛡️ Admin Panel
- Create agents and manage users
- Transfer money to agents
- Monitor all transactions
- Role management system

---

## 🔐 Admin Creation Logic

For security purposes, **admin accounts are not publicly accessible**.

👉 To create an admin account:
- Long press on the app title (**"bKash MFS"**) in the login screen
- Admin creation panel will appear

## 🧰 Tech Stack

- **Kotlin (Android)**
- **Firebase Authentication**
- **Cloud Firestore**
- **MVVM Architecture**
- **Navigation Component**
- **ViewBinding**
